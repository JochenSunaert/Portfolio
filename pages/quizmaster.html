Het project

Dit project is een live quizapp waarin een quizmaster vragen stelt aan spelers, die deze direct beantwoorden. De quizmaster beoordeelt de antwoorden en geeft punten, waardoor de quiz in real-time verloopt. Dankzij WebSockets kunnen quizmaster en spelers naadloos met elkaar communiceren en reageren op vragen en antwoorden.


Gebruikte technologiën

Socket.io - React - CSS animations - JS animations - node.js
