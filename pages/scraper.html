Het project

Bij dit schoolproject was het de bedoeling om een scraper te bouwen die data verzamelt van de Torfs-website. Hiervoor gebruikte ik Puppeteer, een Node.js-bibliotheek voor webscraping. In twee weken tijd leerde ik werken met deze technologie, waarbij ik stap voor stap ontdekte hoe je webpagina's kunt automatiseren en gegevens kunt afnemen. Dit project hielp me om mijn kennis van webscraping en Node.js verder uit te breiden.


Gebruikte technologiën

Puppeteer - Javascript - node.js

Mijn aanpak

Eerst heb ik onderzoek gedaan naar welke scrapers het beste zouden werken voor dit project. Uiteindelijk koos ik voor Puppeteer, omdat deze scraper het gedrag van een echte gebruiker nabootst, wat het gemakkelijker maakt om data van de website te halen. Vervolgens zorgde ik ervoor dat Puppeteer automatisch de cookies accepteerde en de gewenste data kon scrapen. Als extra functionaliteit heb ik toegevoegd dat de gescrapete data gedownload kan worden als een Excel-bestand (CSV) en dat gebruikers zelf kunnen kiezen welke items ze van de site willen ophalen.
